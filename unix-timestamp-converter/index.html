Common mistakes

Seconds and milliseconds are easy to confuse. A 10-digit value is usually seconds, while 13 digits is usually milliseconds.

FAQ

Unix timestamp converter questions.

What timezone is used?

The result includes both local browser time and UTC so you can compare them directly.

Can I enter a date string?

Yes. The converter accepts a parseable date string as well as Unix timestamp values.

What is Unix epoch?

The Unix epoch is January 1, 1970 at midnight UTC, the reference point for Unix timestamps.
