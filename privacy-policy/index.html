Privacy Policy

Local tool processing

Web Utility Desk is designed around browser-based tools. The current tools process input on the page using client-side JavaScript. The site does not provide user accounts, user dashboards, file uploads, or server-side storage for tool input in its current static form.

Analytics and advertising

The site may use privacy-respecting analytics, Google Analytics, Google Search Console verification, advertising services, or affiliate links after production launch. Those services may use cookies, device information, or aggregated usage data according to their own policies. Analytics and advertising integrations should only be added with real production identifiers.

Personal information

The static tools do not require personal information to operate. If a contact form or email channel is added, information sent through that channel should be used only to respond to the request, improve the site, or handle legitimate operational needs.

Security note

Even though tools run locally, users should avoid pasting highly sensitive production secrets, private keys, passwords, customer records, or regulated data into any online tool unless they fully understand the environment and risks.
